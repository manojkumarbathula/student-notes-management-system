# Student Notes Management System

A Flask-based web application to manage notes securely.

## Features

* User Registration & Login
* OTP Verification
* Add / View / Update / Delete Notes
* File Upload & Download
* Search Notes
* Excel Export

## Tech Stack

* Python
* Flask
* MySQL
* HTML
* CSS

## Run Project


pip install -r requirements.txt

python app.py
